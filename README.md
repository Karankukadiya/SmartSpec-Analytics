# SmartSpec Analytics

An end-to-end smartphone market analytics project that combines **web scraping, data analysis, statistical analysis, machine learning, SQL, and Power BI** to uncover pricing trends, brand performance, and market insights.

## Features

*  Web scraping of smartphone listings
*  Data cleaning and preprocessing
*  Exploratory Data Analysis (EDA)
*  Statistical analysis and hypothesis testing
*  Smartphone price prediction using machine learning
*  SQL-based business queries
*  Interactive Power BI dashboard

## Tech Stack

* Python (Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn)
* Selenium & BeautifulSoup
* PostgreSQL / SQL
* Power BI

## Project Workflow

Web Scraping → Data Cleaning → EDA → Statistical Analysis → SQL Analysis → Machine Learning → Power BI Dashboard
